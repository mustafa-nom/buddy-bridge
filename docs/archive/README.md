# Archive: Buddy Bridge (pre-PHISH! pivot)

These documents describe the prior **Buddy Bridge** concept — a 2-player asymmetric co-op consisting of *Stranger Danger Park* and *Backpack Checkpoint*. The project pivoted to **PHISH!** on 2026-04-25.

For the active design, see:
- `../GAMEDESIGN.md` — current game design
- `../PHISH_CORE_LOOP.md` — minute-to-minute gameplay
- `../PHISH_CONTENT.md` — fish catalog
- `../PHISH_MVP_PLAN.md` — hackathon scope
- `../../CLAUDE.md` — agent guidance

The git tag `pre-phish-pivot` (commit `fb65cdb`) preserves the full Buddy Bridge state if you need to recover code or assets.

**Do not treat anything in this folder as current source-of-truth.** Some engineering principles (server authority, RemoteService discipline, no file >500 lines, `init.meta.json` Rojo rule) carry over and are restated in the new `CLAUDE.md`. Everything else — Stranger Danger Park, Backpack Checkpoint, the Guide booth concept, capsule pairing as a core requirement — is superseded.
